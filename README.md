# GuardianBand Firmware

GuardianBand is an ESP32-based wearable fall detection and emergency alert system designed to improve the safety of senior citizens living independently.

The project is being developed using ESP-IDF and modern C++ as a modular embedded firmware application.

## Hardware

- ESP32 (Wemos Lolin D32)
- MPU6050 IMU
- Push Button
- LED
- Buzzer
- Li-ion Battery

## Development Environment

- ESP-IDF
- C++
- CMake
- Git
